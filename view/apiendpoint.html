API Endpoints

User Login

            URI       https://localhost:8080/signup.html
            Method	  POST
            Data	  {
                      "email": "abc@yahoo.com",
                      "password": "123456"
            }
            Result	{
                "firstname": "abc",
                "lastname": "ddd",
                "email": "abc@yahoo.com",
                "password": "123456",
            }
            Error	{
                "login": "fail"
            }
